A modern tourism management web application built using **ASP.NET MVC 5**, designed to promote Pakistan's tourist destinations through an interactive and user-friendly platform. The application enables users to discover destinations, share reviews, and contact the administrators, while providing an admin dashboard for content management.

---
 📖 Overview

Explore Pakistan is a full-stack web application that allows users to:

- Discover tourist destinations across Pakistan
- Search and filter destinations by category
- View detailed information about each destination
- Register and log in securely
- Submit reviews and ratings
- Contact the administration
- Access an admin dashboard for destination management

The project follows the **Model-View-Controller (MVC)** architecture to ensure clean, maintainable, and scalable code.

## 🏗 Architecture

The project follows the **MVC (Model-View-Controller)** architecture.

```
Presentation Layer
        │
Views (HTML, Bootstrap)
        │
Controllers (Business Logic)
        │
Models (Entity Framework)
        │
SQL Server Database
```

## 💻 Installation

1. Clone the repository

```bash
git clone https://github.com/ayesharain/Explore-Pakistan.git
```

2. Open the solution in **Visual Studio 2022**

3. Restore NuGet packages.

4. Configure the SQL Server connection string in `Web.config`.

5. Update the SQL Server connection string in `Web.config`.

6. Build and run the project using Visual Studio 2022.
